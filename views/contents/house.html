하우스 - 인테리어 SNS
팀원 : 기획 ( 김성민 ), 디자인 ( 문석영, 이경훈 ), 개발 ( 이보빈, 박지현, 김소윤, 임주현 )
사용자 : 여성. 20-30대. 주부/신혼
개요

인테리어 시장성과 현재 운영하고 있는 기획자의 인테리어 하우스 블로그의 인원으로 보았을 때,
인테리어 추천, 홍보 효과를 효율적으로 하기 위해선 인테리어 SNS 앱이 필요하였다.
넥스터즈 동아리에서 하우스 앱을 만들게 되었다.
카테고리는 인테리어, 수다톡, 마이페이지가 기본적인 구성으로 되어있다.

개발 - 서버, DB 개발, 안드로이드 (일부) 개발

앱 서버는 Open API 규격에 맞추어 안드로이드 앱과 데이터 통신을 개발하였다.
유저 관리는 모바일 유저에 대한 토큰을 주어 그 토큰으로 로그인 관리하도록 하였다.
DB 단은 인테리어 게시물을 저장할 게시판 DB와 사용자 DB를 만들고, 스크랩, 좋아요 기능에 대한 DB를 설계하였다.
안드로이드에서 앱 서버와 송수신할 때, http통신을 json 형태로 데이터를 전송하고 View로 데이터를 보여주는 역할을 하였다.

Reference

Android - Spring-Android-REST-Template, Universal-image-loader, imageSilder, Kakao, Facebook Open API, Jackson JSON, Joda time
Server - JBoss, Spring Framework, MyBatis, PostgreSQL, JUnit, Jackson JSON, Joda time
